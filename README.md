### Scheduled backups in your menu bar. 'Nough said!

**Shout out** to _Chris Hocking_ of [FCP.cafe](https://fcp.cafe) and Late Nite software for the idea of an overflow drive.

Backup any source to multiple destinations and if it matches an overlimit size it can go to an overflow drive (anywhere really)

See the **README**, User Guide and Workflows guide in repo for more details.
